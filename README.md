
# Microsoft 365 Developer Sandbox – PoC

This repository contains a minimal Proof of Concept showcasing development
and architecture patterns for Microsoft 365, focused on:

- SharePoint Online
- Power BI
- Power Platform (Power Automate)
- Entra ID security model

## Purpose

The goal of this project is to:
- Validate SharePoint Online as a data source
- Test Power BI reporting on top of M365 workloads
- Experiment with permission models and user access
- Prepare foundation for SPFx and Graph API extensions

## Technologies

- Microsoft 365
- SharePoint Online
- Power BI Service & Desktop
- Power Automate
- Microsoft Entra ID

## Status

This is an early architecture and development PoC intended for
use within a Microsoft 365 Developer Program sandbox.
