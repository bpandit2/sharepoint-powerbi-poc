# M365 PoC Architecture

## Overview

The Proof of Concept is based on a standard Microsoft 365 tenant
using SharePoint Online as the primary data storage layer.

## Components

- SharePoint Online Lists as structured data source
- Power BI datasets connected to SharePoint
- Power BI reports embedded in SharePoint pages
- Power Automate flows for basic data processing
- Entra ID used for role-based access control (RBAC)

## Intended Use

This PoC is designed for:
- Development and testing
- Architecture validation
- Learning and experimentation

No production workloads or customer data are planned.
